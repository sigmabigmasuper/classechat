🔧 Come configurare Firebase (5 minuti)

  1. Vai su console.firebase.google.com → Crea progetto (nome libero)
  2. Nel menu laterale clicca Realtime Database → Crea database → scegli Europa
  3. In "Regole" sostituisci tutto con: {"rules":{".read":true,".write":true}} e pubblica
  4. Vai su Impostazioni progetto (icona ⚙️) → Aggiungi app Web (</>)
  5. Copia i valori di firebaseConfig e incollali in cima a questo file HTML
  6. Condividi il file .html con i tuoi compagni — tutti usano lo stesso!
